## Snappy

Command line tool for taking snapshots of a webpage.

### Install

  ```
  $ pip install snapify
  ```


### Usage

Copy the url of the webpage to clipboard and run following commands:

  - to copy the snap url to clipboard

  ```
  $ snapify clip
  ```
  - to open snap in webbrowser
  
  ```
  $ snapify web
  ```
